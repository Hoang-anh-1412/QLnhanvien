## 概要
資格手当（T_資格手当）を表示・管理する画面。資格手当の検索、表示、作成、編集、削除機能を提供する。

- **画面エンドポイント:** `/qualification-allowances`
- **背景色:** 青 (#3B82F6)、メイン表示エリアは白

## 初期表示
1. デフォルト検索条件を設定:
   - 名称: 空欄
   - 等級: すべて
   - 種類: すべて

## 検索実行
**API:** `GET /api/v1/qualifications/allowances`

```typescript
interface QualificationSearchParams {
  keyword?: string;
  type?: string;
  grade?: string;
  employeeCode?: string;
  page?: number;
  size?: number;
  sort?: string;
}
```

- 資格一覧をテーブル形式で表示
- ページング、スクロール、ソート、複数選択をサポート

## 業務機能
1. **資格手当新規作成:** 作成画面に遷移

---

## ユーザー操作フロー
1. 画面を開く → デフォルトフィルター初期化  
2. 検索条件を選択 → "検索"を押す  
3. システムが資格手当一覧を表示  
4. ユーザーが資格手当を選択して:
   - 詳細表示 / 編集 / 削除
   - 関連画面に遷移
